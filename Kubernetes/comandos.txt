kubectl get nodes - lista os nós do cluster
kubectl run NOME_DADO_PRO_POD --image=NOME_IMAGEM - cria um 'pod' (container) com um nome específico e baseado em uma imagem
kubectl get pods - lista os pods criados
kubectl describe pod NOME_POD
kubectl edit pod NOME_DADO_PRO_POD - abre arquivo .txt com arquivos de configuração do pod para edição
